# StatGrid Pro

A modern, animated statistics showcase component built with pure HTML, CSS, and JavaScript. Perfect for displaying company metrics, achievements, and performance indicators with smooth counting animations.

## Live Demo

https://thisislefa.github.io/StatGrid-Pro

## Features

- **Smooth Animations**: Number counters that animate when scrolling into view
- **Fully Responsive**: Adapts beautifully across all device sizes
- **Modern Design**: Clean, professional aesthetics with hover effects
- **Performance Optimized**: Efficient Intersection Observer implementation
- **Accessibility Ready**: Semantic HTML and proper contrast ratios
- **Easy Customization**: Well-structured CSS variables for quick theming

## Tech Stack

- **HTML5** - Semantic markup with BEM methodology
- **CSS3** - CSS Grid, Flexbox, and custom properties
- **JavaScript** - ES6+ with Intersection Observer API
- **Google Fonts** - Inter font family
